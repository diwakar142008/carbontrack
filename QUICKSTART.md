### Change Colors

Edit CSS variables in `:root`:

```css
:root {
  --primary-emerald: #YOUR_COLOR;
  --primary-cyan: #YOUR_COLOR;
}
```

### Add New Sections

1. Create HTML section
2. Add CSS styling
3. Add GSAP scroll trigger
4. Update navigation

### Modify Calculator

1. Update `EMISSION_FACTORS`
2. Edit `calculateEmissions()`
3. Update display values
4. Test calculations

### Add Custom Animations

```javascript
gsap.to(element, {
  property: value,
  duration: 1,
```
